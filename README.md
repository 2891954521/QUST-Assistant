### 简介 📃

- Qust Helper 是一款移动端教务系统客户端，旨在为师生提供一个便捷的教务访问方式，随时随地查看课程安排、考试成绩、教务通知等信息。
- Qust Helper 还是一款优秀的课表软件，能够轻松从教务导入课程信息，自动生成个性化的课表。
- 基于全新的 Jetpack Compose 架构，拥有更加出色的用户体验。*后续可能支持多平台*。
